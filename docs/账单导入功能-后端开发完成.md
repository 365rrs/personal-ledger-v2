## 📝 文件格式要求

### Excel/CSV 表头
| 列名 | 必填 | 类型 | 示例 |
|------|------|------|------|
| 类型 | 是 | 文本 | INCOME 或 EXPENSE |
| 金额 | 是 | 数字 | 1000.00 |
| 分类 | 是 | 文本 | 工资 |
| 描述 | 否 | 文本 | 2025年1月工资 |
| 交易时间 | 是 | 日期时间 | 2025-01-13 10:00:00 |

## 🔧 使用说明

### 1. 启动前准备
```bash
# 1. 确保数据库已创建并执行schema.sql
# 2. 更新application.yml中的数据库配置
# 3. Maven安装依赖
mvn clean install
```

### 2. 启动应用
```bash
mvn spring-boot:run
```

### 3. 访问Swagger文档
```
http://localhost:8080/swagger-ui.html
```

### 4. 测试导入功能
1. 准备Excel或CSV文件（按照格式要求）
2. 调用 `/bill-import/upload` 接口上传文件
3. 调用 `/bill-import/record` 接口查看导入进度
4. 调用 `/bill-import/detail/page` 接口查看明细
5. 调用 `/bill-import/convert` 接口转换为账单

## ⚠️ 注意事项

1. **文件格式**：必须严格按照表头要求，列名必须完全匹配
2. **时间格式**：必须为 `yyyy-MM-dd HH:mm:ss` 格式
3. **交易类型**：只能是 `INCOME` 或 `EXPENSE`
4. **金额**：必须大于0
5. **重复检测**：基于 type + amount + transaction_time + description 计算MD5
