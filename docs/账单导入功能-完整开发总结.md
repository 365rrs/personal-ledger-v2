## ⚠️ 重要提示

### 文件格式要求
1. **表头必须完全匹配**（包括中文）
2. **时间格式**：yyyy-MM-dd HH:mm:ss
3. **交易类型**：INCOME 或 EXPENSE
4. **金额**：必须大于0
5. **文件大小**：不超过10MB

### 数据指纹规则
```
data_hash = MD5(type + amount + transaction_time + description)
```

### 状态流转
```
导入状态：SUCCESS/FAILED
重复状态：UNCHECKED → UNIQUE/DUPLICATE
转换状态：PENDING → CONVERTED/SKIPPED/CONVERT_FAILED
```
